## Your data, your insights

The Valyu Publisher Analytics API gives you programmatic access to the same analytics visible in your [publisher dashboard](https://publisher.valyu.ai). Track usage, monitor earnings, and export detailed reports for your content across the Valyu network.

## How it works

When AI agents query the Valyu search network and your content is returned in the results, that usage is tracked and attributed to your organization. The Publisher Analytics API lets you retrieve this data programmatically.

```
Buyer queries Valyu Search API
        |
        v
Your content returned in results
        |
        v
Usage logged with your org ID (DynamoDB)
        |
        v
Publisher Analytics API <-- You are here
```

## Endpoints

| Endpoint | Description |
|----------|-------------|
| [`POST /analytics/seller-usage`](/api-reference/endpoint/seller-usage) | Time series usage data by source |
| [`POST /analytics/seller-earnings`](/api-reference/endpoint/seller-earnings) | Earnings with period-over-period comparison |
| [`POST /analytics/latest-usage`](/api-reference/endpoint/latest-usage) | Recent retrieval events with document metadata |
| [`POST /analytics/export-analytics`](/api-reference/endpoint/export-analytics) | Bulk export as CSV or JSON |

## Quick example

```bash
curl -X POST https://api.publisher.valyu.ai/v1/analytics/seller-usage \
  -H "Content-Type: application/json" \
  -H "x-api-key: vypub_your_key_here" \
  -d '{"timeScale": "d", "period": 7}'
```

<Note>
  All endpoints require a `vypub_` API key. Create one from the [API Keys page](https://publisher.valyu.ai/user/account/api-keys) in your dashboard.
</Note>
