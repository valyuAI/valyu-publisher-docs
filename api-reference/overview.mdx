## Base URL

```
https://api.publisher.valyu.ai/v1
```

## Authentication

All endpoints require a `vypub_` API key in the `x-api-key` header:

```bash
-H "x-api-key: vypub_your_key_here"
```

Your organization is automatically identified from the key. See [Authentication](/authentication) for details.

## OpenAPI Specification

The full API is described by an [OpenAPI 3.1 specification](/api-reference/openapi.yaml). Import it into Postman, Insomnia, or any OpenAPI-compatible tool.

## Common patterns

### All endpoints use POST

Even read-only queries use `POST` with a JSON body. This allows complex filter parameters without URL encoding issues.

### Errors

All error responses follow the same format:

```json
{
  "error": "Description of what went wrong"
}
```

| Status | Meaning |
|--------|---------|
| `200` | Success |
| `400` | Invalid request parameters |
| `401` | Missing or invalid API key |
| `500` | Server error |

### Rate limits

The API Gateway enforces a global throttle:
- **Sustained rate**: 50 requests/second
- **Burst**: 100 requests
