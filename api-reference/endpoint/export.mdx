---
title: Export
api: POST /analytics/export
description: Export analytics data as CSV or JSON for a date range. Summary or detailed mode.
---

<ParamField body="startDate" type="string" required>
  Start date in `YYYY-MM-DD` format.
</ParamField>

<ParamField body="endDate" type="string" required>
  End date in `YYYY-MM-DD` format. Maximum 366 days from `startDate`.
</ParamField>

<ParamField body="format" type="string" default="csv">
  Output format. One of `csv` or `json`.
</ParamField>

<ParamField body="detailed" type="boolean" default="false">
  When `true`, returns individual retrieval log entries with DOIs. When `false`, returns daily aggregated metrics per source.
</ParamField>

<RequestExample>
```bash cURL (CSV summary)
curl -X POST https://api.publisher.valyu.ai/v1/analytics/export \
  -H "Content-Type: application/json" \
  -H "x-api-key: vypub_your_key_here" \
  -d '{
    "startDate": "2026-03-01",
    "endDate": "2026-03-31",
    "format": "csv"
  }' -o analytics.csv
```

```bash cURL (JSON detailed)
curl -X POST https://api.publisher.valyu.ai/v1/analytics/export \
  -H "Content-Type: application/json" \
  -H "x-api-key: vypub_your_key_here" \
  -d '{
    "startDate": "2026-03-01",
    "endDate": "2026-03-07",
    "format": "json",
    "detailed": true
  }'
```

```python Python
response = requests.post(
    "https://api.publisher.valyu.ai/v1/analytics/export",
    headers={"x-api-key": "vypub_your_key_here", "Content-Type": "application/json"},
    json={
        "startDate": "2026-03-01",
        "endDate": "2026-03-31",
        "format": "csv",
    },
)

with open("analytics.csv", "w") as f:
    f.write(response.text)
```

```javascript JavaScript
const response = await fetch("https://api.publisher.valyu.ai/v1/analytics/export", {
  method: "POST",
  headers: { "x-api-key": "vypub_your_key_here", "Content-Type": "application/json" },
  body: JSON.stringify({
    startDate: "2026-03-01",
    endDate: "2026-03-31",
    format: "json",
  }),
});
const data = await response.json();
```

## Summary mode fields

| Field | Type | Description |
|-------|------|-------------|
| `date` | string | Date (YYYY-MM-DD) |
| `source` | string | Data source identifier |
| `earnings` | number | Earnings for this source on this date (USD) |
| `retrievals` | integer | Number of retrievals |

## Detailed mode fields

| Field | Type | Description |
|-------|------|-------------|
| `timestamp` | string | ISO 8601 timestamp |
| `transaction_id` | string | Unique transaction identifier |
| `source` | string | Data source identifier |
| `earnings` | number | Earnings from this retrieval (USD) |
| `document_count` | integer | Number of documents retrieved |
| `dois` | array | DOIs of retrieved documents |

<Note>
  The response includes a `Content-Disposition` header with a suggested filename, e.g., `valyu-analytics-2026-03-01-to-2026-03-31.csv`.
</Note>
