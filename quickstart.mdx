title: Quickstart
description: Get your first Publisher Analytics API call working in under a minute.
---

<Steps>
  <Step title="Get your API key">
    Log into your [publisher dashboard](https://publisher.valyu.ai) and navigate to **Account > API Keys**.

    Click **Create API Key**, give it a name, and copy the key. It starts with `vypub_`.

    <Warning>
      Store your API key securely. You can reveal it later from the dashboard, but treat it like a password.
    </Warning>
  </Step>

  <Step title="Make your first request">
    Fetch your usage data for the last 7 days:

    <CodeGroup>
    ```bash cURL
    export VALYU_PUB_KEY="vypub_your_key_here"

    curl -X POST https://api.publisher.valyu.ai/v1/analytics/seller-usage \
      -H "Content-Type: application/json" \
      -H "x-api-key: $VALYU_PUB_KEY" \
      -d '{"timeScale": "d", "period": 7}'
    ```

    ```python Python
    import requests

    API_KEY = "vypub_your_key_here"
    BASE_URL = "https://api.publisher.valyu.ai/v1"

    response = requests.post(
        f"{BASE_URL}/analytics/seller-usage",
        headers={
            "Content-Type": "application/json",
            "x-api-key": API_KEY,
        },
        json={"timeScale": "d", "period": 7},
    )

    data = response.json()
    for bucket in data:
        print(f"{bucket.get('day', 'N/A')}: ${bucket.get('All', 0):.4f}")
    ```

    ```javascript JavaScript
    const API_KEY = "vypub_your_key_here";
    const BASE_URL = "https://api.publisher.valyu.ai/v1";

    const response = await fetch(`${BASE_URL}/analytics/seller-usage`, {
      method: "POST",
      headers: {
        "Content-Type": "application/json",
        "x-api-key": API_KEY,
      },
      body: JSON.stringify({ timeScale: "d", period: 7 }),
    });

    const data = await response.json();
    data.forEach((bucket) => {
      console.log(`${bucket.day}: $${bucket.All?.toFixed(4)}`);
    });
    ```
    </CodeGroup>
  </Step>

  <Step title="Explore the response">
    The response is an array of time buckets, each containing earnings broken down by data source:

    ```json
    [
      {
        "day": "2026-03-05T00:00:00",
        "All": 0.0523,
        "source/academic": 0.0312,
        "source/web": 0.0211
      },
      {
        "day": "2026-03-06T00:00:00",
        "All": 0.0891,
        "source/academic": 0.0654,
        "source/web": 0.0237
      }
    ]
    ```

    - **`All`** - total earnings across all sources for this time bucket
    - **`source/xxx`** - per-source breakdown
    - Time key matches your `timeScale` (`hour`, `day`, `week`, or `month`)
  </Step>
</Steps>
